# TOWER-DEFENSE-GAME
Tower defense game made in cpp and sfml
A 2D Tower Defense game built using C++ and SFML as an Object-Oriented Programming (OOP) project. The game features multiple enemy types, tower upgrades, wave-based progression, and a grid-based map system.

#  Features
- Object-Oriented Design (Inheritance, Polymorphism, Abstraction)
- Multiple tower types:
  - Cannon Tower
  - Sniper Tower
  - Machine Gun Tower
  - Slow Tower
  - Bomb Tower
- Multiple enemy types:
  - Basic Enemy
  - Fast Enemy
  - Tank Enemy
  - Flying Enemy
  - Healer Enemy
- Wave-based enemy spawning system
- Gold system for buying/upgrading towers
- Lives system (game over condition)
- Score tracking system
- Tower upgrade system (Level 1 to 3)
- Tower range visualization
- Grid-based map with path system
- Game Over and Win screens

#  Controls

- Left Click → Select tower / Place tower
- Right Click → Upgrade tower
- ESC → Deselect tower
  
#  Game Mechanics

- Enemies follow predefined waypoints
- Towers automatically attack enemies in range
- Each wave becomes harder
- Player must survive all waves
- Game ends when lives reach zero 

#  Technologies Used

- C++
- SFML (Simple and Fast Multimedia Library)
- Object-Oriented Programming (OOP)
  
#  How to Run

1. Install SFML on your system  
2. Open project in Visual Studio
3. Link SFML libraries  
4. Build and run the project
